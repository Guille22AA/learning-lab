# Java Practice

This folder contains Java exercises and small projects created for learning and practice purposes.

## 📁 Contents

### Exercises

Basic programming exercises focused on:

- Control structures
- Functions and methods
- Object-oriented programming fundamentals

### Projects

More complete exercises simulating real-world problems:

- File system management applications
- Database-based applications
- Console-based simulations

## 🎯 Goal

Improve understanding of Java fundamentals and object-oriented programming through hands-on practice.

## 🚀 Notes

Each project or exercise is self-contained and may include:

- Source code
- Comments explaining logic
- Console-based tests
